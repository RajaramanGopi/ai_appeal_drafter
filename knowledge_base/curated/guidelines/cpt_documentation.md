# CPT and Documentation Guidelines (Appeal Reference)

## E/M levels (e.g., 99213 vs 99214)
- **99213**: Office visit, level 3 — typically 15–29 min or moderate complexity (2 of 3: problem-focused history, expanded exam, low-moderate MDM).
- **99214**: Office visit, level 4 — typically 30–39 min or moderate complexity (2 of 3: detailed history, detailed exam, moderate MDM).
- Documentation must support the level billed: history, examination, and medical decision-making (or time when time-based billing is used). When denied for level, appeal with the note and point to the elements that support the level.

## Bundling and NCCI
- **NCCI (National Correct Coding Initiative)** edits define when two codes cannot be billed together (e.g., column 1/2 edits). If a service is bundled, it is not paid separately unless a valid modifier (e.g., 59, XE, XP, XU, XS) applies and is documented.
- When appealing a bundling denial: either show that the codes were correctly unbundled with a modifier and documentation, or correct the claim and resubmit.

## Modifiers
- **25**: Significant, separately identifiable E/M on same day as procedure — documentation must show a distinct E/M service.
- **59 / XE, XP, XS, XU**: Distinct procedural service — use when NCCI allows unbundling; documentation must support that the procedures were distinct.
- **GA, GZ**: Waiver of liability (e.g., ABN); use per CMS rules when applicable.

## Appeal tip
- For coding denials (invalid CPT, procedure inconsistent with diagnosis, bundling): submit corrected claim when the error is clear; otherwise appeal with the medical record and a clear explanation of how the documentation supports the codes and modifiers used.
