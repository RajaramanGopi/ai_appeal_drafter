# Official denial code references (CARC, RARC, group codes)

This project uses **industry-standard** remittance codes. Drafts should align denial language with the EOB/ERA; always verify codes against the payer’s notice and current code sets.

## Claim adjustment group codes (high level)

Common group codes on remittances include **CO** (contractual obligation / payer adjustment), **PR** (patient responsibility), and **OA** (other adjustment). The numeric **CARC** (claim adjustment reason code) explains *why* the adjustment was made. **RARC** (remittance advice remark codes) add narrative detail.

## Authoritative maintenance

- **CARC and RARC** are maintained under the **X12 / Washington Publishing Company (WPC)** process and updated on published schedules. CMS and trading partners implement these updates on Medicare and commercial systems.
- **CMS** publishes Medicare claims processing guidance (including remittance and appeals concepts) in manuals and transmittals on [cms.gov](https://www.cms.gov/).

## Practical use in appeals

- Quote the **exact** CARC/RARC and group code from the remittance or portal denial.
- If the denial is **medical necessity** or **authorization**, tie the narrative to clinical facts and, when applicable, **LCD/NCD** (Medicare) or the payer’s **medical policy** (commercial).
- If the denial is **administrative** (timely filing, missing information, duplicate), address the specific defect with evidence (e.g., submission proof, corrected claim, additional records).

## Payor portals (verify member’s plan)

- **Aetna**: Provider disputes and appeals hub on the official Aetna healthcare professional site.
- **UnitedHealthcare**: Claims, payments, and appeals guidance on UHCprovider.com.
- **Cigna**: Appeals and disputes for healthcare professionals on cigna.com.
- **BCBS**: Plans are **independent**; use the member’s card and EOB for the correct plan’s forms and addresses.
- **Medicare FFS**: Redetermination with the **MAC**; follow CMS appeal levels. **Medicare Advantage**: plan-level reconsideration first, then CMS-regulated external review levels.
