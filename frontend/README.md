# Appeal Drafter — React UI

Enterprise-style SPA: **React 19**, **TypeScript**, **Vite**, **MUI** (layout, forms, alerts).

| Command | Purpose |
|--------|---------|
| `npm install` | Install dependencies |
| `npm run dev` | Vite dev server (proxies `/api` → uvicorn :8000) |
| `npm run build` | Production bundle → `dist/` (served by FastAPI) |
| `npm test` | Vitest + Testing Library |

**Layout:** `src/api/` (HTTP), `src/components/`, `src/hooks/`, `src/theme/`, `src/types/`, `src/utils/`.

See the repository **README.md** (project root) for full setup.
